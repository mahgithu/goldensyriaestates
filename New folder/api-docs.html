Base URL

جميع الطلبات يجب أن تحتوي على Header: Authorization: Bearer YOUR_API_KEY

GET /api/properties

جلب قائمة العقارات مع دعم الفلاتر والتصفح.

Query Parameters:

govstringمعرّف المحافظة (مثال: damascus)
typestringنوع العقار (apartment, villa, land...)
listingstringنوع العرض (sale, rent, invest)
min_pricenumberالحد الأدنى للسعر بالليرة السورية
max_pricenumberالحد الأعلى للسعر
pagenumberرقم الصفحة (افتراضي: 1)
GET /api/properties?gov=damascus&type=apartment&listing=sale&page=1 // Response 200 OK { "success": true, "data": [ { "id": 1, "title": "شقة فاخرة في قلب دمشق", "type": "apartment", "listing": "sale", "gov": "damascus", "price": 450000000, "area": 180, "rooms": 4, "featured": true, "views": 1240 } ], "pagination": { "page": 1, "total": 45, "pages": 5 } }
POST /api/properties

إضافة عقار جديد إلى المنصة.

POST /api/properties Content-Type: application/json // Request Body { "title": "شقة 3 غرف في حلب", "type": "apartment", "listing": "sale", "gov": "aleppo", "city": "الشهباء", "area": 150, "price": 300000000, "rooms": 3, "baths": 2, "desc": "وصف العقار...", "features": ["موقف سيارة", "تكييف"], "agent": "اسم الوكيل", "agentPhone": "0991234567" } // Response 201 Created { "success": true, "data": { "id": 123, ... }, "message": "تم إضافة العقار بنجاح" }
GET /api/contacts

جلب قائمة جهات الاتصال.

GET /api/contacts?type=agent&gov=damascus // Response 200 OK { "success": true, "data": [ { "id": 1, "name": "أحمد الخطيب", "type": "agent", "phone": "0991234567", "gov": "damascus", "rating": 4.9, "deals": 142 } ] }
POST /api/webhooks

تسجيل Webhook URL لاستقبال الأحداث تلقائياً.

POST /api/webhooks Content-Type: application/json { "url": "https://your-app.com/webhook", "events": ["property.created", "contact.inquiry", "property.updated"], "secret": "your_secret_key" } // Response 201 Created { "success": true, "webhook_id": "wh_abc123", "message": "تم تسجيل Webhook بنجاح" }
